# Design Document

## Purpose
This e-commerce platform allows users to browse, search, and purchase products online. Admins can manage products and orders.

## Modules
- User Authentication (Login, Signup)
- Product Browsing
- Cart & Checkout
- Order Management
- Admin Panel

## Technologies
- Python, Django
- Bootstrap, HTML/CSS
- MySQL

## Folder Structure
- /users/
- /products/
- /orders/
